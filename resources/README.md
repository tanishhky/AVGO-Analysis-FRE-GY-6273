# Resources directory

Drop the analyst PDFs (e.g., the contents of `AVGO_Resources.zip` from NYU Bobst Library)
into this folder if you want to keep them with the project.

The Python scripts do not auto-read these files. They are referenced in the LaTeX
report's "Sources cited" section but used only as background research.

Files that typically go here:
- JPMorgan_AVGO_5-Mar-26.pdf
- Mizuho_AVGO_4-Mar-26.pdf
- UBS_AVGO_5-Mar-26.pdf
- WellsFargo_AVGO_4-Mar-26.pdf
- DeutscheBank_AVGO_4-Mar-26.pdf
- Barclays_AVGO_5-Mar-26.pdf
- Bloomberg_Equity_Demo_Tearsheet.pdf
