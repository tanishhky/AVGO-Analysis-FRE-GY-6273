## 2. Prerequisites

- **Python ≥ 3.10** (tested on 3.12)
- **A LaTeX distribution**: TeX Live (Linux/Mac) or MiKTeX (Windows) with `pdflatex`
- **Make** (optional, for the convenience commands)

Recommended Python packages are pinned in `requirements.txt`:

```bash
pip install -r requirements.txt
```

This installs `pandas`, `numpy`, `yfinance`, `matplotlib`, `seaborn`, `openpyxl`, and `scipy`.

## 3. How to run

### Option A — full rebuild from scratch

```bash
# 1. (optional) refresh market data from yfinance
python scripts/01_data_fetch.py

# 2. run the valuation engine — produces CSVs and summary.json
python scripts/02_valuation.py

# 3. generate the 12 figures
python scripts/03_figures.py
cp figures/*.png report/                    # LaTeX needs them next to the .tex

# 4. build the Excel model
python scripts/04_excel_model.py

# 5. compile the LaTeX report (run twice for hyperref refs)
cd report
pdflatex AVGO_Equity_Research_Report.tex
pdflatex AVGO_Equity_Research_Report.tex
```

### Option B — using the Makefile

```bash
make data        # step 1 — refresh data (slow, calls yfinance)
make valuation   # step 2
make figures     # step 3 (also copies PNGs to report/)
make excel       # step 4
make report      # step 5 (pdflatex twice)
make all         # steps 2-5, skipping data fetch
make clean       # remove generated outputs
```

### Option C — recompile the report only

If you just want the PDF and you trust the existing CSVs/figures:

```bash
cd report
pdflatex AVGO_Equity_Research_Report.tex
pdflatex AVGO_Equity_Research_Report.tex
```

## 4. Replacing the placeholder cover images

The cover page uses two image files in `report/`:

- `broadcom_logo.png` — placeholder, currently not embedded in the cover layout but reserved for future use
- `team_photo.png` — placeholder banner (1800 × 400 px) embedded at the bottom of the cover

To use real images:

1. Drop your replacement files at `report/broadcom_logo.png` and `report/team_photo.png`
2. For the team photo, **match the banner aspect ratio** (roughly 4.5 : 1 — e.g. 1800 × 400). Otherwise the photo will distort or push the cover layout.
3. Recompile: `pdflatex AVGO_Equity_Research_Report.tex` (twice)
